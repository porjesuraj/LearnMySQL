* Command to start mongo shell
```sql
    mongo 
```

* Check databases
```sql
show databases;
-- or
show dbs;
```

* Create database
```sql
use dac_db;
```

* To check currently selected database
```sql
db
```

* To drop database
```sql
db.dropDatabase( );
```

* Show the database statistics
```sql
db.stats();
```
* Create collection
```sql
db.createCollection('contacts');
```
* List of collections
```mongo
show collections;
```
